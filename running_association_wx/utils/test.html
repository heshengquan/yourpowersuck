    马拉松基础知识问答

    注：请完成所有试题，才能打印相关项目的大众等级达标证书。

    1.马拉松(Marathon)国际上非常普及的长跑比赛项目，全程距离26英里385码，折合为42.195公里。

    正确答案： 是

    2.在马拉松运动中，身体大量出汗而汗液的主要成分是水，还有钠、钾、氯、镁、钙、磷等矿物质。

    正确答案： 是

    3.报名前需要认真阅读组委会颁布的《竞赛规程》、《报名须知》。

    正确答案： 是

    4.出发前不需要排空大小便。

    正确答案： 否

    5.参赛过程中应听从裁判、志愿者的指挥。

    正确答案： 是

    6.马拉松比赛日前不能熬夜。

    正确答案： 是

    7.比赛中如出现肌肉拉伤，一般应退出比赛，进入救护站进行治疗。

    正确答案： 是

    8.比赛过程中如果有胸闷、气短、无法呼吸的情况，请立刻停止跑步并求助最近的医生或志愿者。

    正确答案： 是

    9.向志愿者表达感谢，是一名优秀的马拉松选手应有的礼仪。

    正确答案： 是

    10.比赛是有中暑的迹象（乏力、大量出汗、口渴、头痛、头晕、眼花、耳鸣、恶心、胸闷等），应立即退赛，并向志愿者或工作人员寻求帮助。

    正确答案： 是

    11.比赛过程中可以随意变道。

    正确答案： 否

    12.参加马拉松比赛前几天一定要进行身体检查。

    正确答案： 是

    13.领取参赛物品过程中，遇到人多时需要排队。

    正确答案： 是

    14.参加马拉松比赛不需要佩戴计时芯片。

    正确答案： 否

    15.穿上跑鞋后，脚感要求舒适，并稍有空隙。

    正确答案： 是

    16.马拉松赛道每5公里要有分段计时点。

    正确答案： 是

    17.报名马拉松但是没有中签，可以私自印制号码布参赛。

    正确答案： 否

    18.饮料中除了水外还包括钠、钾、氯、镁、钙、磷等矿物质及适量含糖物质，补充机体的能量消耗。

    正确答案： 是

    19.为了防止比赛中出现肌肉拉伤，在赛前要做好充分的准备活动，尤其要活动开下肢。

    正确答案： 是

    20.马拉松比赛过程中，合理的配速非常重要。

    正确答案： 是
